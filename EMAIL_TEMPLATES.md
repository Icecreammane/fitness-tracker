## üí° Tips for Email Success

### Best Practices
- **From Name**: Use real name (Ross, not "FitTrack Pro Team")
- **Send Time**: 9am-11am or 6pm-8pm (user's timezone)
- **Frequency**: Max 2 emails/week (don't spam)
- **Unsubscribe**: Always include (legal requirement)
- **Mobile-friendly**: 50%+ users read on phone

### Subject Line Tips
- Keep under 50 characters
- Use emojis sparingly (1 per subject)
- Create curiosity or urgency
- Avoid spammy words ("FREE!!!", "ACT NOW")

### Email Metrics to Track
- **Open Rate**: Target 20-30%
- **Click Rate**: Target 2-5%
- **Conversion Rate**: Target 1-3%

### Testing
- A/B test subject lines
- Test send times (morning vs evening)
- Test copy length (short vs detailed)

### Flask Integration Example

```python
from sendgrid import SendGridAPIClient
from sendgrid.helpers.mail import Mail

def send_welcome_email(user_email, user_name):
    message = Mail(
        from_email='ross@fittrackpro.com',
        to_emails=user_email,
        subject='Welcome to FitTrack Pro!',
        html_content=render_template('emails/welcome.html', name=user_name)
    )
    
    sg = SendGridAPIClient(os.getenv('SENDGRID_API_KEY'))
    response = sg.send(message)
    return response.status_code
```
